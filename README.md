## 🔑 三个核心概念（必读）

### 1️⃣ **背包问题是"母版"**
- 国赛80%的资源分配问题都是背包问题的变体
- 理解背包 → 理解约束建模 → 快速识别问题类型
- **学习文件**：[02_3_背包问题详解.md](./02_问题复杂度理论/02_3_背包问题详解.md)

### 2️⃣ **NP-Hard判定是"分水岭"**
- 能否在多项式时间求精确解，直接决定了算法选择
- **强NP-Hard** ≠ **普通NP-Hard** ≠ **P问题**，三者求解策略截然不同
- **学习文件**：[02_2_强弱NP-Hard判别法.md](./02_问题复杂度理论/02_2_强弱NP-Hard判别法.md)

### 3️⃣ **改进策略胜过"花哨新算法"**
- 没有一个算法天然适配所有问题
- 国赛赢家的秘密：**先诊断问题缺陷 → 再设计针对性改进 → 最后混合融合**
- **学习文件**：[01_4_改良策略速查表.md](./01_算法基础与改进策略/01_4_改良策略速查表.md)

## 🎓 学习成果评估

完成本知识库的学习，你应该能够：

### ✅ 理论层面
- [ ] 快速判断一个优化问题的复杂度（P、NP、NPC、NP-Hard）
- [ ] 区分普通NP-Hard和强NP-Hard，并选择对应算法
- [ ] 根据问题特征设计针对性的改进策略
- [ ] 理解算法的本质缺陷，而不是盲目套用

### ✅ 实战层面
- [ ] 能用GA、PSO、SA、SSA等基础算法求解离散/连续优化问题
- [ ] 能设计混合算法（如GA+SA、PSO+GA交叉）
- [ ] 能用SSA-BP或GA-BP做预测，用NSGA-II做多目标
- [ ] 能在论文中**准确论证为什么选这个改进策略**

### ✅ 论文层面
- [ ] 论文的算法章节有理论深度（不是简单介绍算法）
- [ ] 实验设计科学（对比实验、参数敏感性分析）
- [ ] 能在一页纸内解释清楚"为什么改"和"改了什么"
- [ ] 获得评审老师的"这队对算法的理解很深刻"的评价

## 🏆 用这个库的正确心态

> **这不是一份"背诵手册"，而是一份"理解工具"。**

关键不在于记住有多少算法，而在于：
1. **理解问题** ← 这是第一位的
2. **诊断缺陷** ← GA易早熟→加入精英保留；BP易陷入局部最优→用群智能优化
3. **设计改进** ← 根据缺陷有针对性地改进
4. **论证改进** ← 在论文中清楚地说明"为什么这样改"

如果你能做到这四点，国赛评审就会看出：**这队真正理解了算法，不是在堆砌方法。**
