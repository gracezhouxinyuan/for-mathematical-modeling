### 1. 国赛题型与经典方法

| 题型       | 典型任务           | 一线方法               | 二线方法               | 输出形式                 |
| ---------- | ------------------ | ---------------------- | ---------------------- | ------------------------ |
| 评价类     | 评价、排序、打分   | 熵权、TOPSIS、GRA | RSR、PCA | 排名表、雷达图、权重分析 |
| 预测类     | 时间序列、需求预测 | 回归、ARIMA   | Prophet | 预测曲线、误差表         |
| 优化类     | 分配、排程、路径   | LP/MILP、DP、网络流    | GA、PSO、SA、ACO、DE   | 最优方案、敏感性图       |
| 机理类     | 传播/动力系统      | ODE/PDE、SIR/SEIR      | 参数反演、稳定性分析   | 动态曲线、相图           |
| 分类聚类类 | 结构识别、分组     | KMeans、层次聚类、SVM  | DBSCAN、GMM、RF        | 聚类图、分类指标         |
| 仿真类     | 不确定性推演       | 蒙特卡洛、马尔可夫     | 离散事件仿真    | 分布图、置信区间         |
| 多目标类   | 冲突目标折中       | NSGA-II、MOPSO、MOEA/D | NSGA-III、SPEA2        | Pareto 前沿              |

### 2. 算法

#### 2.1 评价决策

- AHP（层次分析法）
- 熵权法
- TOPSIS（含熵权-TOPSIS）

#### 2.2 统计回归

- 描述统计、假设检验、方差分析
- 线性/多元回归
- 岭回归、Lasso
- Logistic 回归
- 泊松/负二项回归
- 非线性最小二乘拟合

#### 2.3 时间序列与灰色系统

- 移动平均、指数平滑
- AR/MA/ARMA/ARIMA/SARIMA
- 卡尔曼滤波

#### 2.4 数学规划与组合优化

- LP / IP / MILP / QP / NLP
- 动态规划（含状态压缩）
- 最短路：Dijkstra/Floyd/Bellman-Ford
- 最小生成树：Prim/Kruskal
- 匈牙利算法（匹配）
- 最大流/最小费用流

#### 2.5 元启发式与智能优化

- GA、PSO、SA、ACO、DE、TS
- ABC、SSA、GWO、WOA
- 混合算法：GA+SA、PSO+GA、ACO+2-opt

#### 2.6 多目标优化

- NSGA-II/III
- MOPSO
- MOEA/D
- SPEA2
- 指标：HV、IGD、Spread

#### 2.7 机器学习

- KNN、SVM/SVR
- 决策树、随机森林
- GBDT、XGBoost、LightGBM
- BP、LSTM、Transformer（数据足量时）

#### 2.8 聚类与异常检测

- KMeans、层次聚类
- DBSCAN、GMM
- LOF、Isolation Forest

#### 2.9 插值与数值计算

- 线性、样条、拉格朗日、牛顿插值
- 数值积分、数值微分

#### 2.10 概率与随机过程

- 二项、泊松、正态分布
- 马尔可夫链
- 排队论
- 蒙特卡洛

#### 2.11 机理与微分方程

- ODE/PDE
- SIR/SEIR/SEIRD
- 参数估计、灵敏度分析、情景仿真

### 4. 论文写作系统

#### 4.1 章节结构

1. 问题重述
2. 模型假设
3. 符号说明
4. 模型建立
5. 求解方法
6. 结果分析
7. 模型评价
8. 结论与建议

#### 4.2 算法段落固定模板

- 原法简介（3-5 行）
- 原法不足（本题语境）
- 改进策略（1-3 点）
- 实验验证（基线 + 改进）
- 结论（是否显著提升）

#### 4.3 图表最低要求

- 每个关键模型至少 1 张核心图。
- 每个实验至少 1 张对比表。
- 图中必须有坐标、单位、图例。

---

### 5. 代码工程规范

- 每个脚本必须函数化，保留 `if __name__ == "__main__":` 入口。
- 变量名语义化，不使用无意义单字符变量（循环索引除外）。
- 每个算法模块至少包含：`输入`、`求解`、`输出`、`可视化`。
- 每次改良必须保留 baseline 对照文件。

---

### 6. 数据处理与实验规范

- 缺失值：删除/插值/模型填补（记录选择理由）
- 异常值：IQR/3σ/模型法（记录阈值）
- 归一化：MinMax 或 Z-score（记录公式）
