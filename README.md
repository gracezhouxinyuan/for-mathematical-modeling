### 1. 项目使命

#### 1.1 核心目标

- 建立国赛常见问题的一站式方法库与代码库。
- 实现“看得懂原理、写得出代码、讲得清论文、做得完复盘”。
- 每个算法沉淀为“理论卡片 + 模板代码 + 案例结果 + 论文段落”。

#### 1.2 成果定义

完成后应具备以下能力：

- 30 分钟内完成题型判断与初始选型。
- 2 小时内搭建可运行 baseline 模型。
- 1 天内完成改良版本与对比实验。
- 1-2 天内形成国赛风格完整小论文。

### 2. 国赛题型全景与方法映射

| 题型       | 典型任务           | 一线方法               | 二线方法               | 输出形式                 |
| ---------- | ------------------ | ---------------------- | ---------------------- | ------------------------ |
| 评价类     | 评价、排序、打分   | AHP、熵权、TOPSIS、GRA | 模糊综合评价、RSR、PCA | 排名表、雷达图、权重分析 |
| 预测类     | 时间序列、需求预测 | 回归、ARIMA、GM(1,1)   | LSTM、XGBoost、Prophet | 预测曲线、误差表         |
| 优化类     | 分配、排程、路径   | LP/MILP、DP、网络流    | GA、PSO、SA、ACO、DE   | 最优方案、敏感性图       |
| 机理类     | 传播/动力系统      | ODE/PDE、SIR/SEIR      | 参数反演、稳定性分析   | 动态曲线、相图           |
| 分类聚类类 | 结构识别、分组     | KMeans、层次聚类、SVM  | DBSCAN、GMM、RF        | 聚类图、分类指标         |
| 仿真类     | 不确定性推演       | 蒙特卡洛、马尔可夫     | 离散事件仿真、Agent    | 分布图、置信区间         |
| 多目标类   | 冲突目标折中       | NSGA-II、MOPSO、MOEA/D | NSGA-III、SPEA2        | Pareto 前沿              |

### 3. 算法总库

#### 3.1 评价决策

- AHP（层次分析法）
- 熵权法
- TOPSIS（含熵权-TOPSIS）
- 灰色关联分析（GRA）
- 模糊综合评价
- RSR（秩和比）
- VIKOR（可选）

#### 3.2 统计回归

- 描述统计、假设检验、方差分析
- 线性/多元回归
- 岭回归、Lasso
- Logistic 回归
- 泊松/负二项回归
- 非线性最小二乘拟合

#### 3.3 时间序列与灰色系统

- 移动平均、指数平滑
- AR/MA/ARMA/ARIMA/SARIMA
- 卡尔曼滤波
- GM(1,1)、GM(1,N)、Verhulst

#### 3.4 数学规划与组合优化

- LP / IP / MILP / QP / NLP
- 动态规划（含状态压缩）
- 最短路：Dijkstra/Floyd/Bellman-Ford
- 最小生成树：Prim/Kruskal
- 匈牙利算法（匹配）
- 最大流/最小费用流

#### 3.5 元启发式与智能优化

- GA、PSO、SA、ACO、DE、TS
- ABC、SSA、GWO、WOA
- 混合算法：GA+SA、PSO+GA、ACO+2-opt

#### 3.6 多目标优化

- NSGA-II/III
- MOPSO
- MOEA/D
- SPEA2
- 指标：HV、IGD、Spread

#### 3.7 机器学习

- KNN、SVM/SVR
- 决策树、随机森林
- GBDT、XGBoost、LightGBM
- BP、LSTM、Transformer（数据足量时）

#### 3.8 聚类与异常检测

- KMeans、层次聚类
- DBSCAN、GMM
- LOF、Isolation Forest

#### 3.9 插值与数值计算

- 线性、样条、拉格朗日、牛顿插值
- 数值积分、数值微分

#### 3.10 概率与随机过程

- 二项、泊松、正态分布
- 马尔可夫链
- 排队论
- 蒙特卡洛

#### 3.11 机理与微分方程

- ODE/PDE
- SIR/SEIR/SEIRD
- 参数估计、灵敏度分析、情景仿真

### 5. 算法选型系统

#### 5.1 快速规则

- 评价排序：AHP/熵权/TOPSIS/GRA
- 线性约束优化：LP/MILP
- 大规模组合：GA/ACO/TS
- 连续参数优化：PSO/DE
- 多目标冲突：NSGA-II/MOPSO
- 动态传播：SIR/SEIR + 参数估计

#### 5.2 判别逻辑

- 是否线性：线性优先规划法。
- 是否强组合爆炸：优先元启发式。
- 是否多目标：优先 Pareto 框架。
- 是否高噪声小样本：优先灰色/统计稳健方法。

### 6. 论文写作系统

#### 6.1 章节结构

1. 问题重述
2. 模型假设
3. 符号说明
4. 模型建立
5. 求解方法
6. 结果分析
7. 模型评价
8. 结论与建议

#### 6.2 算法段落固定模板

- 原法简介（3-5 行）
- 原法不足（本题语境）
- 改进策略（1-3 点）
- 实验验证（基线 + 改进）
- 结论（是否显著提升）

#### 6.3 图表最低要求

- 每个关键模型至少 1 张核心图。
- 每个实验至少 1 张对比表。
- 图中必须有坐标、单位、图例。

---

### 7. 代码工程规范

- 每个脚本必须函数化，保留 `if __name__ == "__main__":` 入口。
- 变量名语义化，不使用无意义单字符变量（循环索引除外）。
- 每个算法模块至少包含：`输入`、`求解`、`输出`、`可视化`。
- 每次改良必须保留 baseline 对照文件。

---

### 8. 数据处理与实验规范

#### 8.1 数据处理

- 缺失值：删除/插值/模型填补（记录选择理由）
- 异常值：IQR/3σ/模型法（记录阈值）
- 归一化：MinMax 或 Z-score（记录公式）

#### 8.2 实验规范

- 固定随机种子
- 统一评价指标
- 统一训练/验证划分
- 每个改良至少 3 组参数实验

### 11. 算法学习地图（60天）

#### 阶段一：基础建模能力（Day 1-15）

- 评价：AHP、熵权、TOPSIS、GRA
- 统计回归：线性回归、Logistic、泊松
- 预测：ARIMA、GM(1,1)

#### 阶段二：优化主干能力（Day 16-35）

- LP/MILP、DP、最短路、网络流
- TSP/VRP 基础模型
- 参数敏感性分析

#### 阶段三：智能优化能力（Day 36-50）

- GA、PSO、SA、ACO、DE
- 混合优化与改良策略
- 多目标 NSGA-II/MOPSO

#### 阶段四：综合实战能力（Day 51-60）

- 机理模型（SIR/SEIR）
- 综合题复现（至少 2 套）
- 论文标准化输出
