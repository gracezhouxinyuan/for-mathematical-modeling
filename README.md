## 💡 建议的学习路径

### 📅 **4周快速入门路线**

```
Week 1: 基础算法 (01系列)
  Day 1-2: 遗传算法(GA) → 看改良策略
  Day 3-4: BP神经网络 → 看群智能优化方案
  Day 5-7: 动态规划 → 看降维处理

Week 2: 问题分析 (02系列)
  Day 8-9: NP-Hard基础分类
  Day 10-12: 四步诊断法 + 实战演练
  Day 13-14: 背包问题 + 距离度量

Week 3: 算法全景 (03系列)
  Day 15-16: 进化算法 vs 群智能对比
  Day 17-18: SSA、ACO等新算法
  Day 19-21: 多目标优化 + 混合策略

Week 4: 实战训练 (04系列)
  Day 22-23: 经典问题模板(TSP/VRP)
  Day 24-26: 历年真题分析
  Day 27-28: 论文写作 + 代码整理
```

### 📅 **12周深度冲刺路线**

```
Weeks 1-3: 第一阶段 (基础算法) 深度学习
  - 每个算法配2-3个实战练习
  - 代码调试到"开箱即用"

Weeks 4-6: 第二阶段 (问题分析) 精通诊断
  - 分析10+道历年真题的问题属性
  - 能独立判断问题的复杂度分类

Weeks 7-9: 第三阶段 (算法全景) 建立代码库
  - 完成GA、PSO、SA、SSA、ACO各一个实现
  - 完成SSA-BP、GA-BP两个混合实现
  - 完成NSGA-II多目标框架

Weeks 10-12: 第四阶段 (实战案例) 套题训练
  - 做3-5套历年真题完整解答
  - 形成个人的论文模板、代码模板、建模流程
```

## 🎓 学习成果评估

完成本知识库的学习，你应该能够：

### ✅ 理论层面
- [ ] 快速判断一个优化问题的复杂度（P、NP、NPC、NP-Hard）
- [ ] 区分普通NP-Hard和强NP-Hard，并选择对应算法
- [ ] 根据问题特征设计针对性的改进策略
- [ ] 理解算法的本质缺陷，而不是盲目套用

### ✅ 实战层面
- [ ] 能用GA、PSO、SA、SSA等基础算法求解离散/连续优化问题
- [ ] 能设计混合算法（如GA+SA、PSO+GA交叉）
- [ ] 能用SSA-BP或GA-BP做预测，用NSGA-II做多目标
- [ ] 能在论文中**准确论证为什么选这个改进策略**

### ✅ 论文层面
- [ ] 论文的算法章节有理论深度（不是简单介绍算法）
- [ ] 实验设计科学（对比实验、参数敏感性分析）
- [ ] 能在一页纸内解释清楚"为什么改"和"改了什么"
- [ ] 获得评审老师的"这队对算法的理解很深刻"的评价

## 📝 如何使用这个库进行每日学习

### 1️⃣ **每天学习新内容**
- 选择上面的学习路线，按文件顺序学习
- 遇到不理解的概念，用 Ctrl+F 在当前文件搜索，或查看相关文件的交叉引用

### 2️⃣ **每天补充新笔记**
- 在 [05_每日学习记录/学习日志模板.md](./05_每日学习记录/学习日志模板.md) 中记录
- 格式统一，便于后续复习和知识管理

### 3️⃣ **遇到好的案例或想法**
- 直接补充到对应的文件中（如发现了GA的新改进方法，就补到 01_1_遗传算法(GA).md）
- 保持原有案例的结构和风格，使用同样的 Markdown 格式

### 4️⃣ **定期复习与思考**
- 每周抽出1-2小时整理一次学习笔记
- 用"一句话总结"的方式记录每个算法的核心思想
- 按复杂度、应用场景重新分类整理
