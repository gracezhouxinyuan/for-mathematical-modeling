# logistic_regression 数据与来源说明

- 本专题包使用 sklearn 合成二分类数据，用于演示逻辑回归在不平衡场景下的基线与改良效果。
- 真实题目可替换为用户行为、风险判定、故障识别、是否违约等二分类数据。
