# descriptive_statistics 数据与来源说明

- 本专题包的数据为合成示例，刻意加入缺失值和异常值，用于展示描述统计与稳健统计的差异。
- 真题或真实数据可替换为企业运营、人口统计、考试成绩、监测指标等表格型数据。
- 推荐先检查缺失、异常值和偏态，再做均值、标准差、中位数、四分位数与分布图。
