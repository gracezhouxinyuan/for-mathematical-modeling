# nonlinear_least_squares 数据与来源说明

- 本专题包数据为合成指数衰减曲线，用于演示非线性最小二乘拟合的基线与改良流程。
- 真题中常见的拟合对象包括增长曲线、衰减曲线、剂量反应曲线、蒸发/衰减过程、S 型扩散曲线等。
